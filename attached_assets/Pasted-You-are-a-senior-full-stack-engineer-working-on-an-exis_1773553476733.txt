FEATURE 1 — COURSE REVIEWS & RATINGS

Students must be able to review a course after enrollment.

Capabilities

• Rate course from **1 to 5 stars**
• Write optional review text
• Each user can review a course only once
• Allow updating the review
• Display average rating and total reviews on course page

Database Table

## course_reviews

id (PK)
user_id (FK users.id ON DELETE CASCADE)
subject_id (FK subjects.id ON DELETE CASCADE)
rating INT CHECK (rating BETWEEN 1 AND 5)
review_text TEXT
created_at
updated_at

Unique constraint

(user_id, subject_id)

Indexes

(subject_id)
(rating)

}
]
